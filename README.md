# Pinacotheca

Photogallery. This is just an example code.

### Requirements
 - PHP >= 5.4 
 - MySQL >= 5.6
 - PDO 

### Supported frameworks
 - Flight 1.3 
 - Medoo 1.7
 - Smarty 3.x
 - WideImage 1.x

### Installation

1. Clone this repository:
```sh
$ git clone https://github.com/double-null/pinacotheca.git ./
```
2.  Set up the project dependencies using the composer:
```sh
$ composer install
```
3. Import file **pinacotheca.sql** into MySQL.
4. Edit the database configuration in the file **settings/database.php** .
